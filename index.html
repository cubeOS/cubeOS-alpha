CubeOS-Alpha

Still in pre-alpha!

CubeOS is an open-source operating system for DCPU-16. It is developed by the CubeOS Organization, and features UNIX-like shells, a standard filesystem, and a package system, so software is very simple to add.

Site Remodel

01/07/12

In preparation for the upcoming "release" of CubeOS, I, (Sasha Crofter,) have remodeled this site. CubeOS has always been available, and therefore cannot be "released" in the conventional sense, but it is nearing a "full version," which contains all or most of the features expected of an operating system. A demo will be coming soon.
